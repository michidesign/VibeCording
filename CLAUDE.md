# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

複数の個人プロジェクトを収容するモノレポ構成。各プロジェクトは独立しており、共通の依存関係やビルドシステムは存在しない。

## プロジェクト一覧

| ディレクトリ | 技術スタック | 概要 |
|-------------|-------------|------|
| `photo-selector/` | Python (OpenCV, CustomTkinter) | 写真品質評価・7段階分類ツール |
| `task_app/` | 単一HTML (Vanilla JS) | Webベースタスク管理アプリ |
| `kids-sunglasses-app/` | HTML/JS (face-api.js) | 顔検出サングラス合成アプリ |
| `flag_game/` | 単一HTML | 国旗ゲーム |
| `man_hour_calculator/` | 単一HTML | 工数計算ツール |
| `sunglasses_app/` | HTML/JS | サングラスアプリ（旧版） |
| `90_Archives/` | - | アーカイブ済みプロジェクト |


## アーキテクチャ特性

- **日本語UI**: すべてのアプリケーションは日本語インターフェース

## CLAUDE.mdの作成ルール

### 言語設定
- 各プロジェクトディレクトリに作成される`CLAUDE.md`は**必ず日本語**で記述すること
- プロジェクトの説明、技術仕様、使用方法などすべてのセクションを日本語で記載
- コード例のコメントも日本語で記述
- 英語の技術用語や固有名詞はそのまま使用可（例: React, npm, API）

### 推奨構成
各プロジェクトの`CLAUDE.md`には以下のセクションを含めることを推奨:
1. **プロジェクト概要** - アプリケーションの目的と機能
2. **技術スタック** - 使用言語・フレームワーク・主要ライブラリ
3. **ディレクトリ構成** - 主要ファイル・フォルダの説明
4. **セットアップ方法** - インストール・起動手順
5. **主要機能** - 実装されている機能の詳細
6. **開発時の注意事項** - コーディング規約や設計上の制約
