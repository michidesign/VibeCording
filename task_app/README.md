# My Tasks

シンプルで美しいWebベースのタスク管理アプリケーション。Apple Design Systemに準拠した洗練されたUIで、直感的なタスク管理を実現します。

## 特徴

- **単一ファイル構成** - `mytasks.html` のみで動作。サーバー不要、依存関係なし
- **Apple風デザイン** - ミニマルで洗練されたUI、ガラスモーフィズム、滑らかなアニメーション
- **ダークモード対応** - システム設定に連動、手動切り替えも可能
- **完全オフライン** - localStorageでデータを永続化
- **レスポンシブ** - デスクトップ、タブレット、モバイルに対応

## 機能

### タスク管理
- タスクの作成・編集・削除
- カテゴリとステータスによる分類
- 期限の設定（自由形式または日付ピッカー）
- メモの追加
- ドラッグ&ドロップによる並び替え

### サブタスク
- 各タスクにサブタスクを追加可能
- サブタスクごとに期限・メモを設定
- 個別の完了管理

### ビュー切り替え
| ビュー | 説明 |
|--------|------|
| Active | 進行中のタスク |
| Archived | 完了したタスク |
| Trash | 削除したタスク（復元可能） |

### フィルタリング
- カテゴリでフィルタ
- ステータスでフィルタ
- 組み合わせフィルタ

### カスタマイズ
- カテゴリの追加・編集・削除
- ステータスの追加・編集・削除（色付き）
- グローバルメモ欄

### データ管理
- JSONエクスポート/インポート
- 自動バックアップ（`_json/` ディレクトリ）

## 使い方

### 起動
`mytasks.html` をブラウザで直接開くだけで使用できます。

```bash
# macOS
open mytasks.html

# Windows
start mytasks.html

# Linux
xdg-open mytasks.html
```

### 基本操作

1. **タスク追加**: サイドバーの入力フォームに情報を入力し「追加」ボタン
2. **タスク編集**: タスクカード上で直接編集（タイトル、カテゴリ、ステータス等）
3. **タスク完了**: 「完了」ボタンでアーカイブに移動
4. **タスク削除**: 「削除」ボタンでゴミ箱に移動
5. **並び替え**: ドラッグハンドル（⋮⋮）をドラッグ

### キーボードショートカット
- タスク名入力後 `Enter` でタスク追加

## バックアップと復元

### エクスポート
1. サイドバー下部の「Export」ボタンをクリック
2. JSONファイルがダウンロードされる

### インポート
1. サイドバー下部の「Import」ボタンをクリック
2. 以前エクスポートしたJSONファイルを選択

### 自動バックアップ
`_json/` ディレクトリにバックアップファイルが保存されます。
形式: `mytasks_backup_YYYYMMDDHHMM.json`
